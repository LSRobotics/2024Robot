# 2024Robot

 Robot that will be used for 2024 FRC Season

 
## Controls:

### Driver Controller:

- **Swerve Zero Heading** = Y button
- **Intake** = A button to run, B button to clear
- **Elevator** = D-Pad Up and Down buttons

### Operator Controller:

- **Distance Shot** = A button
- **Short Shot** = B button
- **Pass to Shooter** = Right Trigger
